Proyectos

SERVIDOR DE ECOMMERCE

API Rest de ecommerce realizada con NestJS, TypeScript, TypeORM, PostgreSQL.

El proyecto maneja rutas públicas y privadas. Manejo de roles users y admin.

REGLAS DE NEGOCIO

  • PRODUCTOS: El usuario no registrado puede visualizar los productos.
  • REGISTRO: El usuario puede registrarse en el sistema, utilizo Bcrypt para la encriptación de las contraseñas.
  • INICIO DE SESIÓN: El usuario puede iniciar sesión, utilizo JWT para el manejo de tokens.
  • USUARIO: El usuario registrado puede visualizar y modificar su información y eliminar su perfil.
  • CARRITO DE COMPRAS: El usuario puede agregar productos al carrito de compras.
  • ORDENES: El usuario puede visualizar sus órdenes con el detalle de los productos comprados y el precio total.
  • PRODUCTOS: El usuario registrado como administrador puede registrar, editar y eliminar productos.
  • USUARIOS: El usuario registrado como administrador puede visualizar y eliminar usuarios.

TURNERO DIGITAL

Proyecto FULLSTACK realizado con ExpressJS, TypeScript, TypeORM, PostgreSQL.

Este proyecto permite a los usuarios registrarse, iniciar sesión y también, solicitar y cancelar turnos. Posee una vista de administrador que permite visualizar todos los usuarios con sus turnos, y puede bloquear y desbloquearlos.

FUNCIONALIDADES DE USUARIOS

  • Registrarse e iniciar sesión. Utilizo nodemailer para la confirmación de correo.
  • Visualizar y modificar su perfil, pudiendo subir una foto.
  • Solicitar un turno y cancelarlo.

FUNCIONALIDADES DE ADMINISTRADOR

  • Visualizar todos los usuarios con sus turnos.
  • Bloquear y desbloquear usuarios.
  • Agregar servicios.
